# Dataset Description

## Primary Dataset
### Air Quality Index (AQI)
- Grain: Day × City × State
- Time Period: 2022–2025
- Key Metrics: AQI, PM2.5, PM10, NO2, SO2

## Secondary Datasets

### Health Data
- Grain: Week × State × District × Disease
- Metrics: Reported cases and deaths
- Focus: Respiratory and pollution-related diseases

### Vehicle Registration Data
- Grain: Month × State × Vehicle Class × Fuel Type
- Used to analyze EV adoption and pollution proxy

### Population Data
- Grain: Year × State × Gender
- Metric: Urban population projections (2011–2036)

## Purpose
These datasets together enable:
- Market sizing
- Health impact correlation
- Product feature prioritization
- City-level risk scoring
